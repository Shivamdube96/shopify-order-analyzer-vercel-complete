# Shopify Order Analyzer — Vercel Ready

This repo is ready-to-deploy on Vercel (Hobby plan friendly). It builds a fully client-side analyzer.

## Quick Start
```bash
npm install
npm run dev
```

## Deploy (Vercel)
- Push to GitHub → Vercel → New Project → Import → Deploy.
- Or use CLI:
```bash
npm i -g vercel
vercel
vercel --prod
```
